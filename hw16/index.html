 Задание 16 - jQuery
Разработайте скрипт, который получает из input введенный url адрес. Адрес обрабатывается таким образом, что введенные имена приводятся к виду: Ввели
   • http://primer.ua
   • http://www.primer.ua
   • https://primer.ua
Получили: primer (удаляются протоколы и локальные зоны) Затем введенная строка обрабатывается по правилам:
   • все символы о или О заменяются на 0 (нуль)
   • l заменяются на единицу (символ эль на единицу)
   • все символы «i» заменяются на 3
   • все буквы s заменяются на 5
   • к результирующей строке добавляется случайный набор из 4-х символов.
Результат – выводится на экран.
Для реализации использовать jQuery.
